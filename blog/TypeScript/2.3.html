                        # Array е’Ң Tuple

                        Typescript ж–ҮжЎЈең°еқҖпјҡArray е’Ң Tuple (opens new window)

                        # Array

                        //гҖҢзұ»еһӢ + ж–№жӢ¬еҸ·гҖҚжқҘиЎЁзӨәж•°з»„пјҡ
let arrOfNumbers: number[] = [1, 2, 3, 4]
//ж•°з»„зҡ„йЎ№дёӯдёҚе…Ғи®ёеҮәзҺ°е…¶д»–зҡ„зұ»еһӢпјҡ
//ж•°з»„зҡ„дёҖдәӣж–№жі•зҡ„еҸӮж•°д№ҹдјҡж №жҚ®ж•°з»„еңЁе®ҡд№үж—¶зәҰе®ҡзҡ„зұ»еһӢиҝӣиЎҢйҷҗеҲ¶пјҡ
arrOfNumbers.push(3)
arrOfNumbers.push('abc')

//ж•°з»„жіӣеһӢ Array < type >
let arrOfNumbers: Array<number> = [1, 2, 3];

                        # Tuple

                        // е…ғзҘ–зҡ„иЎЁзӨәе’Ңж•°з»„йқһеёёзұ»дјјпјҢеҸӘдёҚиҝҮе®ғе°Ҷзұ»еһӢеҶҷеңЁдәҶйҮҢйқў иҝҷе°ұеҜ№жҜҸдёҖйЎ№иө·еҲ°дәҶйҷҗе®ҡзҡ„дҪңз”Ё
let user: [string, number] = ['viking', 20]

//дҪҶжҳҜеҪ“жҲ‘д»¬еҶҷе°‘дёҖйЎ№ е°ұдјҡжҠҘй”ҷ еҗҢж ·еҶҷеӨҡдёҖйЎ№д№ҹдјҡжңүй—®йўҳ
user = ['molly', 20, true] // зј–иҜ‘жҠҘй”ҷ
